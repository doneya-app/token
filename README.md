# Token Universe Explorer

A visual exploration tool for understanding tokens, tokenization, and how text is processed by language models.

## Project Info

**URL**: https://token.kristoffer.dev
**Repository**: https://github.com/doneya-app/token

## Local Development

To run this project locally, you'll need Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Clone the repository
git clone https://github.com/doneya-app/token.git

# Navigate to the project directory
cd token

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project is deployed on GitHub Pages at [token.kristoffer.dev](https://token.kristoffer.dev).

To deploy changes, push to the main branch and GitHub Actions will automatically build and deploy the site.
